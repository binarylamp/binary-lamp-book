# Binary Lamp Style Guide

## Mission

Making IT and English accessible to non-engineers.

Building a book in public.

## Audience

- Non-engineers
- IT beginners
- Students
- Professionals learning IT English

## Writing Style

- Write mainly in English.
- Use simple, natural English.
- Keep paragraphs short.
- Explain the original meaning before the IT meaning.
- Avoid translation-like English.
- Add Japanese notes at the end.

## Workflow

News
→ Select one word
→ Create a draft
→ Human review
→ Publish

## Goal

Build a practical IT English book through continuous public writing on GitHub.
